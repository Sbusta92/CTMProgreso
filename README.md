# CTM Progreso

Aplicación full-stack desarrollada como Trabajo de Fin de Grado (TFG) para la gestión de torneos de tenis de mesa.

## Tecnologías utilizadas

### Backend
- Java 21
- Spring Boot
- Maven
- MariaDB

### Frontend
- React Native
- Expo
- JavaScript

## Funcionalidades

- Gestión de usuarios
- Creación de torneos
- Organización automática por divisiones
- Registro de resultados de partidos
- Ascensos y descensos automáticos
- Historial de ligas ganadas
- API REST para comunicación entre frontend y backend
